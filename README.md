# MAM-cli

Tanglechat with your friends.

## Installation

Clone:
```
git clone https://github.com/chrisdukakis/mam-cli.git

cd mam-cli

```

Install dependencies:
```
npm install
```

Start it:
```
npm start
```

## Usage

- Enter your seed
- Publish messages with `pub` command
- Get channels with `get`
- Subscribe to channels with `sub`
